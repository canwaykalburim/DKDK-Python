## A. 기본 문법 익히기

### **I ) 함수**

- **정의**  

함수는 '하나의 특별한 목적의 작업을 수행하기 위해 독립적으로 설계된 코드의 집합'이라고 정의할 수 있다.  
> EX ) Print( )

**함수명 (Fonction name)**  
함수의 이름  

**인자 값 (Argument value)**  
함수가 동작하기 위해서 넣어야하는 값

**반환 값 (return value)**  
함수가 실행된 후 되돌려주는 값

- **정리**

***print('Hello, World!')***  

함수명: print  
인자 값: 'Hello, World'  
반환 값: 없음

- **참고**  

함수에서 함수명은 반드시 필요하지만 인자 값과 반환 값은 필수적이지 않다.  
또한 반드시 1개일 필요도 없다.

### **II ) 변수**

- **정의**

숫자, 문자와 같은 값들을 저장하는 이름을 가진 공간이다.

> 책에서 변수는 프로그램이 실행할 때 필요한 **데이터의 값을 저장한 곳을 가리키는 이정표**라고 설명한다. 이는 변수는 메모리상의 주소를 사람이 읽을 수 있는 쉬운 언어로 표현하기 위한 용도로 사용되기 때문이다.

**변수명 (Value name)**  
문자 그대로 변수의 이름이다. 변수의 이름을 정하는 것은 생각보다 중요하며, 간단하고 본인의 의도가 잘 드러나는 변수명을 작명하는 것이 좋다.

**변수명 유의사항**  
변수명을 만들 때에는 반드시 지켜야 하는 규칙이 몇 가지 있다.

1. 첫 글자는 반드시 영문 대소문자 혹은 언더바(_)로 시작한다.

2. 나머지 글자들은 영문자, 숫자 혹은 언더바(_)로 구성한다.

3. 대소문자를 구분한다.

4. 길이에 대한 제약은 없다.

5. 아래의 예약어는 변수명으로 사용할 수 없다.

~~~ Python
False    class       finally    is          return
None     continue    for        lambda      try
True     def         from       nonlocal    while
and      del         global     not         with
as       elif        if         or          yield
assert   else        import     pass
break    except      in         raise
~~~

위의 규칙은 변수명을 짓기 위한 최소한의 규칙이다.  
그렇다면 좋은 변수명은 무엇일까?

**좋은 변수명 만들기**

1. 짧게 축약하기보다는 의미가 담겨 있는 것이 좋다.
> EX ) 'stay_here'을 'sh'처럼 쓰지 않는 것이 좋다.
 
2. 변수명의 길이를 너무 길게 쓰지 않는다.

3. 변수명 형식을 일관성 있게 쓴다.
> EX ) stay_here과 stayHere을 섞어 쓰지 않는다. 파이썬은 소문자와 언더바가 섞여있는 형식을 기본으로 한다고 한다.

5. 언더바로 시작하는 병수명은 특별한 경우에만 사용한다고 한다.

- **정리**

***my_name = input('이름')***

변수: my_name

변수명을 규칙에 맞게 쓰고 좋은 변수명을 만들어 쓴다.

- **참고**  

변수 자체가 물리적인 공가늘 제공하는 것은 아니다.  
변수는 메모리상의 데이터를 쉽게 참조하기 위한 라벨과도 같다.

### **IV ) 블록과 들여쓰기**  

- **정의**

**들여쓰기**는 문장을 쓸 때 문자열을 오른쪽으로 당겨 주변의 문장과 구분하는 것이다.  
가독성을 개선하기 위해 소스 코드를 형식화하는 데 쓰인다.

**블록**은 마치 한 문단처럼 보이는 코드의 한 부분을 뜻한다.  

- **들여쓰기와 블록 예시**

예제는 책에 나온 예시를 사용하였다.

**C 예시**

~~~ C
int main(void)
{
    for (int i = 0; i < 10; i++)
    {
        printf("%d", i);
        printf("위 숫자를 제곱하면");
       printf("%d", i * i);             // 오류 미발생
    }
    return 0;
}
~~~

**Python 예시**

~~~Python
for x in rage(10):
    print(x)
    print('위 숫자를 제곱하면')
   print(x**2)                  # 들여쓰기로 인한 오류 발생
~~~

C : 세미콜론(;) 때문에 들여쓰기로 인한 오류가 없다.  
그리고 블록문 선언은 중괄호({ })로 시작한다.

Python : 들어쓰기로 인한 오류가 발생하므로 들여쓰기를 잘 해야한다.  
그리고 블록문 선언은 콜론(:)으로 시작한다.

- **정리**

파이썬에서 들여쓰기는 블록문을 선언하는데 중요한 역할을 하므로 일관성 있는 들여쓰기를 해야한다.

기본 들여쓰기는 공백 4칸이다.

콜론(:)으로 블록문을 선언한다.

- **참고**

**들여쓰기**  
Python, Haskell, occam과 같은 프로그래밍 언어에서는 들여쓰기로 프로그램 구조를 해석하기도 한다고 한다.

**블록**  
블록은 중괄호로 묶여 있는 경우가 많고 보통 1개 이상의 명령어를 가지고 있으나 주석으로 이루어진 블록이나 아무 내용도 없는 빈 블록도 가능하다.

블록문 안에 또 다른 블록문을 사용하는 것도 가능하나 너무 많은 블록문의 중첩은 소스를 읽기 불편하게 만들 수 있으므로 피해야한다.

### **V ) 클래스와 객체**

클래스와 객체는 이 파트만으로 책 한 권을 낼 수 있을 만큼 이해하기 어렵기에 여기서는 클래스와 객체에 대해 책과 인터넷을 참고해 설명할 것이다.

- **정의**

**클래스**  
**클래스(Class)** 는 **객체 지향 프로그래밍(OOP)** 에서 특정 객체를 생성하기 위해 변수와 메소드를 정의하는 일종의 틀이다.  
객체를 정의 하기 위한 **상태(멤버변수)** 와 **메서드(함수)** 로 구성된다.

**객체**  
컴퓨터 과학에서 **객체(Object)** 는 저장공간에서 할당되어 값을 가지거나 식별자에 의해 참조되는 공간을 의미하며, **변수(Variable)**, **자료 구조(Data structure)**, **함수** 또는 **메소드**가 될 수 있다고 한다.

- **책에서 설명한 클래스와 객체**

참고한 책은 "조인석"님의 파이썬 프로그래밍입니다.

**객체**  
프로그래밍을 한다는 것은 마치 사람의 머리 속에 있는 세상을 컴퓨터로 옮기는 작업과 크게 다르지 않다. 그렇다 보니 **실제 세상을 본따서 컴퓨터 내부에 새로운 세상을 창조하기 위해 태어난 개념**이 '**객체'** 라고 할 수 있다.  

**객체의 특징**  
객체는 크게 세 개의 특징을 가지고 있다. 바로 **식별자**, **상태** 그리고 **행위**이다. 이는 다시 말하면 객체를 구별해 주는 **아이디**, 상태를 보여주는 실제 **데이터 값** 그리고 행위를 실행하는 **함수**로 얘기할 수 있다.

**클래스**  
객체는 실제로 메모리에 상주하고 있는 생명체라고 표현 한다면 클래스는 **이 생명체를 만들기 위한 일종의 명세서**이다. 마치 요리를 만들 때 어떤 재료들을 어느 정도 양을 가지고 무슨 순서로 요리를 해야 할지 적어 놓은 레시피와도 같다.

- **참고**  
파이썬의 모든 구현체는 클래스들로 이루어져 있으며 실제로 동작할 때 모든 기능은 객체에 의해 수행된다.

간혹, 클래스와 객체를 혼동할 수 있지만 하나의 클래스에 여러 개의 객체가 생성될 수 있다는 점을 기억하면 쉽게 구분할 수 있다.

- **클래스와 객체 사용 예시**

~~~ Python
>>> Ex1 = '클래스와 객체'
>>> Ex2 = '클래스와 객체'
~~~

이 변수들이 곧 객체로 같은 문자열을 넣었지만 메모리 상에서는 서로 다른 객체로 존재한다.

~~~ Python
>>> Ex1 == Ex2  # 변수의 데이터 값 비교하기
True
>>> Ex1 is Ex2  # 객체 비교하기
False
~~~

두 개의 등호(==)는 데이터 값 자체를 비교한다.  
'is'는 같은 객체 여부를 판단하는 명령어이다.

~~~ Python
>>> id(EX1)         # 객체 식별자 확인
>>> 1901452942048
>>> id(Ex2)         # 객체 식별자 확인
>>> 1901452942192
~~~

파이썬 기본 내장 함수 id()를 통해 **객체의 식별자**를 확인할 수 있는데, 위에 결과와 같이 값이 서로 다른 것을 확인할 수 있다. (식별자는 같은 컴퓨터라도 실행할 때 마다 달라진다.)

~~~ Python
>>> type(ex1)
<class 'str'>
>>> type(ex2)
<class 'str'>
~~~

type() 함수를 활용하면 **어떤 클래스가 사용되었는지 확인**할 수 있는데, 두 변수 모두 'str'이라는 클래스 타입이라는 것을 알 수 있다.

## B. 데이터형

- **정의**

데이터 형이란 사람이 프로그래밍 언어를 사용하여 데이터의 공통된 특징과 용도에 따라 분류하여 정의한 것이다.

### **I ) 숫자, 논리, 문자열형**

**숫자형 (Numberic type)**  

데이터를 더하고, 빼고, 곱하고 나눌 수 있는 데이터형

- 정수형  
  
파이썬에서 정수는 int로 나타낸다.
> EX ) 4, 8, -9

- 실수형

파이썬에서 실수는 float로 나타낸다.
> EX ) 4.8, -9.48

- 참고

파이썬에서는 복소수를 표현하는 complex형 등을 제공힌다.

- 연산 기호

~~~Python

# x 더하기 y
x + y
EX ) 4 + 8 = 12

# x 빼기 y
x - y
EX ) 15 - 3 = 12

# x 곱하기 y
x * y
EX ) 3 * 6 = 12

# x 나누기 y
x / y
EX ) 72 / 6 = 12

# x를 y로 나눈 값의 내림 값
x // y
EX ) 186 // 15 = 12

# x를 y로 나눈 나머지
x % y
EX ) 38 % 13 = 12

# x 부호 변경
-x
EX) 12 => -12

# x 변경 없음
+x
EX ) 12 => +12

# x의 y승
x**y
EX ) 12 ** 12 = 8916100448256
~~~

**논리형 (Boolean Type)**

데이터 중 참과 거짓을 통하여 표현할 수 있는 데이터형

파이썬에서는 참은 **True**, 거짓은 **False**로 표시한다.  
참고로 앞 문자가 대문자이다.

- 사용 예시

~~~ Python
# x가 y보다 클 떄

>>> x < y   # x가 y보다 작은가?
False
>>> x > y   # x가 y보다 큰가?
True
>>> x == y  # x와 y가 같은가?
False
>>> x != y  # x와 y가 다른가?
True
~~~

- 비교 연산자

~~~Python
<       # 작은
<=      # 작거나 같은
>       # 큰
>=      # 크거나 같은
==      # 같은
!=      # 다른
is      # 같은 객체
is not  # 같은 객체가 아닌
~~~

- 논리 연산자

~~~Python
x or y  # x, y 둘 중에 하나라도 참이면 참, 나머지는 거짓
# => or는 x가 거짓인 경우에만 y를 수행한다.
x and y # x, y 모두 참이면 참, 나머지는 거짓
# => and는 x가 참인 경우에만 y를 수행한다.
not x   # x가 참이면 거짓, x가 거짓이면 참
# => not은 논리 연산자가 아닌 연산자에 비해 우선순위가 낮다.
~~~

- 사용 예시

~~~Python
# x > y > z 일 때
>>>  x < y or y > z
True
>>> x > y and x < z
False
>>> not x < y
True
~~~

- 참고

논리형인 True와 False는 숫자형에 속해 있다.  
True는 1, False는 0을 나타낸다.

**문자열 형(String Type)**  

데이터가 여러 문자로 구성되어 있고 다른 문자와 연결될 수 있으며 데이터에 포함된 문자열의 길이를 확인할 수 있는 데이터형  

파이썬에서 문자열 데이터형은 **str**이라고 하며 입력하고 싶은 문자열을 홀 따옴표(')로 감싸거나 쌍 따옴표(")로 감싸면 문자열로 인신한다.  

여러 줄인 경우 홑 따옴표 세 개나 쌍 따옴표 세 개로 감싸면 된다.

> EX ) print ('''  
> 이렇게  
> 사용합니다  
> ''')

- 사용 예시

~~~ python
>>> '안녕하세요'
'안녕하세요'
>>> "안녕히계세요"
"안녕히계세요"
~~~

- 참고

다음의 상황에서는 에러가 뜬다.

~~~ python
>>> 'I'm doctor'
SyntaxError: invalid syntax
~~~

이는 중간의 홑 따옴표를 출력을 위한 문자로 인식하기 때문이다.  
이럴 때는 크게 두 가지 방법으로 해결 가능하다.  

1. 전체 문자열을 쌍 따옴표로 감싼다.
2. 문자로 표기하고 싶은 기호 앞에 역 슬래시(\\)를 입력한다.

다음과 같이 사용한다.

~~~ python
>>> "I'm doctor"
"I'm doctor"
>>> 'I\'m doctor'
"I'm doctor"
~~~
